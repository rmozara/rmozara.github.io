This website serves as a personal workspace and archive — a place to collect essays, research notes, and creative work that reflect ongoing inquiries into science, philosophy, and Integral Yoga.

Technology

  • Built entirely with static HTML, CSS, and lightweight JavaScript
  • Custom modular layout — no frameworks, no CMS
  • Designed for simplicity, clarity, and longevity

Design Philosophy

The design aims for a quiet, stable rhythm — minimal distractions, readable margins, and consistent proportions. Every page follows the same grid logic and type hierarchy for a calm, unified flow.
